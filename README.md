# Yang's Craft Agent Skills Collection

这是一个 Craft Agent 技能集合，包含了各种实用的 AI 技能。

## 📋 技能列表

### 核心技能
- `skill-flow-yang-000` - 技能流程管理
- `ddl-yang-002` - DDL 任务管理
- `easy-yang-017` - 简单任务处理

### 学习技能
- `concept-mastery-yang-008` - 概念掌握
- `pre-reading-yang-004` - 预读学习
- `qateaching-yang-005` - QA 教学
- `upgrade-yang-006` - 知识升级

### 生活技能
- `money-yang-018` - 财务管理
- `foggdesign-yang-001` - 行为设计
- `relationship` - 人际关系

### 工作技能
- `work-explain-yang-011` - 工作说明
- `logic-yang-016` - 逻辑分析
- `file-storage-yang-010` - 文件管理

### 工具技能
- `list-making-yang-009` - 列表制作
- `information-showing-yang-007` - 信息展示
- `paper-collecting-yang-013` - 论文收集
- `jiantu-tuijian-yang-014` - 简图推荐

### 知识星球技能
- `zsxq-group` - 知识星球小组
- `zsxq-note` - 知识星球笔记
- `zsxq-shared` - 知识星球分享
- `zsxq-topic` - 知识星球话题
- `zsxq-user` - 知识星球用户

### 其他技能
- `dokobot-yang-003` - Dokobot 技能
- `gpt-image` - GPT 图像处理

## 🚀 使用方法

1. 克隆此仓库到本地
2. 将技能目录复制到你的 Craft Agent 技能目录
3. 在 Craft Agent 中使用对应的技能命令

## 📝 技能格式

每个技能目录包含：
- `SKILL.md` - 技能定义文件（必需）
- `icon.svg` 或 `icon.png` - 技能图标（推荐）
- 其他资源文件（可选）
